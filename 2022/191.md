# 记录一下字符串分隔split各语言之间的不同

今天才发现不同编程语言对字符串的 `split` 是有差距的

## js

```js
'a:b:c:d'.split(':', 2); // 输出结果为 ['a','b']
```

是先把所有的都拆分成数组，然后取前面的几个

## c# 

```c#
"a:b:c:d".Split(":", 2); // 输出结果为 ['a','b:c:d']
```

是拆分前几个，剩余的都原样放在最后一个

## php

```php
explode(':', 'a:b:c:d', 2); // 输出结果为 ['a','b:c:d']
```
